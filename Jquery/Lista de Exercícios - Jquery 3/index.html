1. Observe a instrução abaixo. Quando a página é carregada temos que mostrar uma mensagem ao usuário para que ele saiba como reservar uma viagem. Para isso, vamos criar um nó <span> com nosso número de telefone (55-47-8888-4567- jQuery-aerolinhas para reservar esta viagem) atribuindo este valor a uma variável (JavaScript). Como você inicializaria a variável mensagem?

var mensagem = $("<span> 55-47-8888-4567-jQuery-aerolinhas </span>")

2. Adicione o número do telefone imediatamente antes (before()) do botão "RESERVE AGORA".

$(document).ready(() => {
$(".book").before(mensagem)
})

3. Pensando bem, seria melhor acrescentar o nodo <span> ao final do elemento “.usa”.

$(document).ready(() => {
$("li.usa").append(mensagem)
})

4. Agora remova o botão "RESERVE AGORA" até que possamos implementá-lo posteriormente.

$(document).ready(() => {
$("li.usa>.book").remove()
})

5. Agora insira um manipulador de eventos para que possamos remover um botão quando este for clicado.

$(document).ready(() => {
$("li.usa>.book").on("click", () => {
$("li.usa>.book").remove()
});
})

6. Alterar o manipulador do clique (evento) para ser acionado quando o item da lista “.tour” for clicado.

$("li.tour").on("click", () => {
$("li.usa>.book").remove()
});
